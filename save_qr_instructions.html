🏧 GCash InstaPay QR Code Setup

To use your InstaPay QR code in the laundry management system, please follow these steps:

Step 1: Save the QR Code Image

Right-click on your InstaPay QR code image and select "Save image as..."

Step 2: Use the Correct Filename

Save the image with this exact filename:

gcash_qr_instapay.jpg

Step 3: Save to the Correct Location

Save the file in this folder:

c:\xampp\htdocs\Mar\mar\lab\uploads\

Step 4: Verify

The complete file path should be:

c:\xampp\htdocs\Mar\mar\lab\uploads\gcash_qr_instapay.jpg

✅ Once saved correctly, the QR code will automatically appear in the GCash payment modal!

Note: The InstaPay QR code you provided can be scanned by any banking app or GCash for payments.
